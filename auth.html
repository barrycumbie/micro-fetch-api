Flow: This public page logs in, stores session in sessionStorage, then redirects to index.html.

Auth Page

This page is public. It calls POST /api/authn/login with the shared demo password.

  • If you are already signed in, you are redirected to index.html.
  • If login succeeds, token + user display name are saved in sessionStorage.
  • Then you are redirected to index.html.

Want to test the guard? Try opening a protected page first. If you are not logged in, it should redirect back to this auth page.
